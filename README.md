# For My Love ❤️

A romantic website for someone special.

## About

This is a heartfelt website created to express love and share beautiful memories. It includes interactive elements, personal messages, and a timeline of special moments.

## Features

- Interactive timeline of your shared story
- Photo gallery with special memories
- Interactive "girlfriend simulator" 
- Personal letters section
- Countdown timer
- Music player
- Confetti animations
- Responsive design

## Deployment

This project is configured for deployment on Vercel. Simply connect your GitHub repository to Vercel and it will deploy automatically.

### Manual Deployment Steps

1. Fork this repository
2. Connect to Vercel via GitHub
3. Deploy!

## Local Development

To run locally:

```bash
npm install -g serve
serve .
```

Or simply open `index.html` in your browser.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Canvas API for animations
